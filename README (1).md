# 🏠 House Price Prediction using Machine Learning

A simple Machine Learning project that predicts house prices using
**Flask** and **Linear Regression**.

## Features

-   Flask web application
-   Linear Regression model
-   HTML/CSS frontend
-   House price prediction
-   Easy to run in VS Code

## Project Structure

``` text
House-Price-Prediction/
├── app.py
├── train_model.py
├── house_data.csv
├── model.pkl
├── requirements.txt
├── README.md
├── config.json
├── templates/
│   └── index.html
└── static/
    └── style.css
```

## Installation

``` bash
pip install -r requirements.txt
python train_model.py
python app.py
```

Open: http://127.0.0.1:5000
